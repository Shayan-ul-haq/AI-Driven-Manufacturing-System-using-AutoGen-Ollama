# AI-Driven-Manufacturing-System-using-AutoGen-Ollama
# AI‑Driven Manufacturing System (AutoGen + Ollama)

This project demonstrates how **AI agents** can automate decision‑making inside a manufacturing environment.  
Instead of fixed rules or human scheduling, the system uses **LLM‑powered agents** that communicate in natural language to negotiate processing times and award a production contract.

---

## 🚀 Project Overview

This project simulates a **smart manufacturing cell** where:

- **Machine Agents** behave like real factory machines (Drilling, Milling, etc.)
- **Order Manager Agent** acts like a production planner
- Agents communicate using **natural language**
- The system is fully **AI‑driven**, meaning:
  - Machines propose processing times
  - The manager compares bids
  - The best machine automatically receives the contract

This shows how LLMs can replace traditional rule‑based scheduling with **autonomous negotiation**.

---

## 🧠 AI‑Driven Concept

In a normal factory:
- Humans check machine load  
- Humans negotiate lead times  
- Humans decide which machine gets the job  

In this AI‑driven system:
- Machines = AI agents  
- Manager = AI agent  
- Decision‑making = **fully autonomous**  
- Communication = **natural language**  
- Contract awarding = **LLM‑based reasoning**

This is a practical example of how **AI agents can control a smart factory**.
